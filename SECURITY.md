# Security Policy

## Supported Versions

Only the latest published release of `recut-ai` receives security fixes.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report security issues privately via GitHub's built-in private vulnerability
reporting: navigate to the **Security** tab of this repository and click
**Report a vulnerability**. You will receive a response within 5 business days.

## Disclosure Policy

Once a fix is available and released, the vulnerability will be publicly
disclosed in the GitHub release notes with a CVE reference if applicable.
